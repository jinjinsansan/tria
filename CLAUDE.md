# tria Japan Salon

日本一triaを学べる無料オンラインサロン

## 技術スタック
- Next.js 14 (App Router)
- Supabase (認証・DB)
- TailwindCSS + shadcn/ui
- Claude API (AI補助)
- Vercel

## ドキュメント
- `docs/SPECIFICATION.md` - 完全仕様書
- `docs/DATABASE.sql` - Supabase SQL
- `docs/CLAUDE_CODE_PROMPT.md` - 開発ガイド

## 現在のフェーズ
Phase 1: 基盤構築

## 開発ルール
- TypeScript必須
- shadcn/ui使用
- ダークテーマベース
- グラデーション（紫〜青〜緑）をアクセントに

## 主要機能
1. ユーザー認証（メール/パスワード）
2. tria解説コンテンツ
3. フィード（Q&A）
4. オートプレースメント
5. AI補助（X投稿・note記事生成）
6. 管理者パネル
